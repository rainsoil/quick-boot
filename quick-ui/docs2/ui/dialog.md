## Attributes

| 属性名       | 说明           | 类型      | 可选值                           | 默认值  |
| ------------ | -------------- | --------- | -------------------------------- | ------- |
footer | 是否打开fooder | Boolean  |  true/false    | true
mode | 弹框模式 | String |   "dialog", "drawer" |   dialog
visible | 显示或者隐藏 | Boolean  |   |
modalProps | 配置  |  Object |    |
