## Attributes

| 属性名       | 说明           | 类型      | 可选值                           | 默认值  |
| ------------ | -------------- | --------- | -------------------------------- | ------- |
value | 绑定的值 | String/Array  |     |
params | 请求参数,配合url使用 | Object |    |   {}
url | 接口url | String  |   |
label | 按钮名称(必填) |  String |    |
dictType|  字典值type | String |    |
dataList| 字典数据 |  Array  |   | []
remote| 是否加载远程方法 | Boolean | true/false | false
