# 安装
:::danger 安全前必读
`c7-ui` 是基于 `element-plus` 进行的二次封装，所以在使用前请先安装 `element-plus`。

:::
## 安装 element-plus
`Element Plus`安装 👉 [点击前往](https://element-plus.org/zh-CN/guide/installation.html)

## 安装 c7-ui
```shell
# npm
npm install c7-plus --save

# yarn
yarn add c7-plus

# pnpm
pnpm add c7-plus
```
