QForm ​

快捷表单，旨在优化element-plus表单开发过程的代码冗余，配置繁琐等问题。使用QForm，您可以快速完成表单页面的构建以及数据的收集、验证和提交。目前QForm已支持 输入类、选择类、 文本类 等19种的组件。

基础用法 ​

显隐控制 ​

通过表单配置项 formOption 下的vIf,可以实现表单项的显隐控制。

vIf 类型为 boolean 或 一个返回boolean值方法， true 时显示，false 时隐藏。

ts
{
  ...,
  vIf: (vals: any) => {
    // vals 为表单收集的数据对象
      return true
  }
}
// 或
{
  ...,
  vIf: true
}
